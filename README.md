# To-Do List

Projeto simples de lista de tarefas para fins de estudo e portfólio.

## Funcionalidades
- Adicionar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

## Tecnologias
- HTML, CSS, JavaScript

## Como usar
1. Clonar o repositório
2. Abrir o arquivo `index.html` em um navegador
